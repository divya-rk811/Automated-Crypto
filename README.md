# Automated-Crypto
A robust Python-based tool designed to track cryptocurrency price fluctuations in real-time. This project demonstrates proficiency in **API Integration**, **Object-Oriented Programming (OOP)**, and **Automation**.

## Key Features
- **Real-time Data:** Fetches live price feeds using the CoinGecko REST API.
- **Custom Alerts:** User-defined price thresholds with visual status indicators.
- **Error Resilience:** Implemented exception handling for network timeouts and API rate limits.
- **Clean Architecture:** Built using a Class-based approach for scalability.

##  Tech Stack
- **Language:** Python 3.x
- **Libraries:** Requests (HTTP handling), Time (Automation)
- **API:** CoinGecko Public API
